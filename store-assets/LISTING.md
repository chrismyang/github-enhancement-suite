## Detailed description

Sharper editing for GitHub, right inside the boxes you already use.

GitHub Enhancement Suite improves GitHub's own comment and description editors in place — it
does not replace them. Your @-mentions, #-references, emoji autocomplete, image upload,
Markdown preview, and ⌘-Enter submit all keep working exactly as before.

WHAT YOU GET

• List-aware indenting — Tab and Shift+Tab indent and outdent the current list item (or
  selection) the way a real editor does, instead of tabbing away from the box. Press Enter and
  the next bullet, number, or task-list checkbox continues automatically; Shift+Enter adds a
  soft line break inside the same item.

• Monospace editor font — Markdown textareas render in a clean monospace font, so tables,
  code, and indentation line up while you type.

• Issue-link search (Ctrl+;) — search issues and pull requests without leaving the box. A
  small panel opens at your cursor; type a query, pick a result, and the reference is inserted
  for you. Searches run against GitHub on your own session — no third-party servers.

• Quick-edit (press 'e') — hover any of your own comments or descriptions and press 'e' to
  flip it straight into GitHub's native edit mode. No reaching for the "…" menu.

PRIVACY

This extension collects no data and sends nothing to any third party. It runs only on
github.com and talks only to GitHub, using your existing logged-in session. There are no
accounts, no analytics, and no tracking.

HOW IT WORKS

The extension enhances GitHub's native textareas using standard, undo-safe edits, so a single
Ctrl/⌘-Z reverses anything it does — and if GitHub ever changes its layout, it quietly falls
back to default behavior rather than breaking the box.
